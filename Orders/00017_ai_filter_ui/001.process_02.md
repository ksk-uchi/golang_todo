# UI挙動・デザイン修正の手順

## 背景と目的

レビューを通じてAIFilterBarに以下の問題があることが判明しました。これらを修正します。

1. **バッジ内クリアボタンが反応しない**: shadcn/ui の Badge コンポーネントはデフォルトで直下の子要素SVGに対し `pointer-events-none` を付与するため、Xアイコンのクリックイベントが発火しない状態になっていました。
2. **アクティブクエリのアイコンデザイン修正**: アクティブフィルターバッジ内のアイコン（Sparkles と X）のサイズを大きくし、Sparkles を塗りつぶし（`fill`）にします。
3. **入力フォーカスの奪取**: `PopoverTrigger` を入力枠を囲む親要素として用いていたため、初回のクリック時に Popover 側がデフォルト動作でフォーカスを奪い、Inputが入力状態になりませんでした。

※ **注意**: スクリプト実行前に、ユーザーが自ら手動で `AIFilterBar.tsx` の Input および内部アイコン周りにスタイル修正（`h-13`, `rounded-full`, `w-6 h-6` など）を加えています。それらの修正は絶対に上書き（元に戻すなどの破壊）をせず、温存して改修を行ってください。

## Task 1: AIFilterBar の Popover 挙動修正（入力フォーカス維持）

### 対象ファイル

`frontend/src/components/todo/AIFilterBar.tsx`

### 変更内容

1. `import { PopoverTrigger } from "@/components/ui/popover";` の部分に `PopoverAnchor` を追加します。不要となる `PopoverTrigger` は削除してください（例: `import { Popover, PopoverContent, PopoverAnchor } from "@/components/ui/popover";`）。
2. `Popover` コンポーネント内の `<PopoverTrigger asChild>` を `<PopoverAnchor asChild>` に置き換えます。
3. `<Input>` 要素に `onFocus={() => setIsPopoverOpen(true)}` と `onClick={() => setIsPopoverOpen(true)}` を追加します。これでInputをクリックまたはフォーカスした際に確実にドロップダウンが開くようになります。
4. `<PopoverContent>` に `onOpenAutoFocus={(e) => e.preventDefault()}` を追加します。これにより開いた瞬間に Input からフォーカスが強制的に外れる現象を防止します。

## Task 2: アクティブフィルター表示（Badge）および機能の修正

### 対象ファイル

`frontend/src/components/todo/AIFilterBar.tsx`

### 変更内容

`{activeFilter && ( <Badge ...> ... </Badge> )}` の部分を以下のように修正します。

1. **Sparkles アイコンの修正**:
   - `Badge` 直下だと `[&>svg]:size-3` の影響を受けて小さくなるのを避けるため `span` 等で囲みます。
   - クラスを `w-5 h-5 text-blue-500` とし、`fill="currentColor"` 属性を指定して塗りつぶしてください。
   ```tsx
   <span className="flex items-center">
     <Sparkles className="w-5 h-5 text-blue-500" fill="currentColor" />
   </span>
   ```
2. **X アイコン（クリアボタン）の修正**:
   - バツ印をクリックしてクリア処理が走るように、`<X>` を単独で置くのではなく `<button>` 要素で囲みます（これにより SVG への `pointer-events-none` の影響を回避し、クリックを有効化します）。
   - クラスを `w-5 h-5` とし、ボタンに元のonClickやホバースタイルを付与します。
   ```tsx
   <button
     type="button"
     className="flex items-center justify-center cursor-pointer hover:text-destructive transition-colors ml-1 focus:outline-none"
     onClick={() => {
       onClearFilter();
       setInputValue("");
     }}
   >
     <X className="w-5 h-5" />
   </button>
   ```
