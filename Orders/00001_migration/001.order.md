# Ent を使った ToDo テーブルのスキーマを作成

## 背景

- Go 言語の学習のため ToDo 管理アプリを作成しようとしている
- バックエンドは Golang + Echo + Ent + TypeSpec というスキルセットを考えている
- `go run -mod=mod entgo.io/ent/cmd/ent new Todo` として ToDo スキーマを作成した
  - `backend/ent/schema/todo.go` というファイルが生成された

## 対応内容

1. ToDo スキーマを更新する
   - ToDo テーブルが下記の形になるように
     - table_name: todo
     - column
       - id
         - int
         - primary_key
         - auto_increment
       - title
         - varchar(64)
         - not null
       - description
         - varchar(256)
         - default ""
         - not null
       - done_at
         - datetime
         - nullable
         - default null
       - created_at
         - datetime
         - not null
         - default now
       - updated_at
         - datetime
         - not null
         - default now
         - on update now
